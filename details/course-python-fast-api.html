Python dengan FastAPI

Bangun backend API modern dan efisien menggunakan FastAPI. Kursus ini cocok untuk kamu yang sudah memahami dasar Python dan ingin membangun RESTful API dengan performa tinggi dan dokumentasi otomatis.

Video Pembelajaran

Apa yang akan kamu pelajari:

  • Pengantar FastAPI dan keunggulannya
  • Struktur project FastAPI
  • Routing dan parameter (path, query, request body)
  • Validasi data dengan Pydantic
  • Response model dan status code
  • Integrasi database dengan SQLAlchemy
  • Autentikasi & otorisasi JWT sederhana
  • Menjalankan aplikasi dengan Uvicorn
  • Testing endpoint API
  • Dokumentasi otomatis (Swagger & ReDoc)
