JavaScript untuk Pemula

Pelajari dasar-dasar JavaScript secara menyeluruh mulai dari konsep variabel hingga manipulasi DOM. Kursus ini dirancang untuk kamu yang ingin mulai membuat website interaktif tanpa perlu pengalaman coding sebelumnya.

Video Pembelajaran

Apa yang akan kamu pelajari:

  • Apa itu JavaScript dan cara kerjanya di browser
  • Variabel, tipe data, dan operator
  • Struktur kontrol: if/else, switch, loop
  • Fungsi dan scope
  • Array dan objek
  • DOM (Document Object Model) dan event handling
  • Manipulasi elemen HTML dengan JavaScript
  • Validasi form sederhana
  • Dasar debugging di browser
