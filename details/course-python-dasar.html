Python Dasar

Kuasai dasar bahasa Python dengan cara yang mudah dan praktis. Kursus ini merupakan titik awal sempurna bagi pemula yang ingin terjun ke dunia pemrograman.

Video Pembelajaran

Apa yang akan kamu pelajari:

  • Pengantar Python dan cara instalasinya
  • Sintaks dasar dan komentar
  • Tipe data: string, integer, float, boolean
  • Operasi dasar dan logika
  • Struktur kontrol: if/else, while, for
  • Fungsi dan modul
  • List, tuple, dictionary, dan set
  • Error handling (try/except)
  • File input/output dasar
  • Praktik mini-project sederhana
